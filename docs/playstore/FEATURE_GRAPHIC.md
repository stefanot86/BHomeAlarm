# Feature Graphic - BHomeAlarm

## Specifiche Google Play Store

### Dimensioni Richieste
- **Larghezza**: 1024 px
- **Altezza**: 500 px
- **Formato**: PNG o JPEG (24-bit, no alpha)
- **Dimensione max file**: 1 MB

---

## Design Proposto

### Layout

```
+------------------------------------------------------------------+
|                                                                  |
|  [LOGO]     BHomeAlarm                    [ICONA CASA]           |
|                                           [CON SCUDO]            |
|             Controlla il tuo                                     |
|             allarme Bticino                                      |
|             dal telefono                   [MOCKUP              |
|                                             TELEFONO            |
|  [ICONA SMS]  [ICONA NOTIFICA]             CON APP]             |
|                                                                  |
+------------------------------------------------------------------+
```

### Elementi Grafici

1. **Logo App** (a sinistra)
   - Icona BHomeAlarm stilizzata
   - Dimensione: circa 120x120 px

2. **Testo Principale**
   - Font: Roboto Bold o Google Sans
   - Testo: "BHomeAlarm"
   - Sottotitolo: "Controlla il tuo allarme Bticino dal telefono"
   - Colore: Bianco o colore primario

3. **Icone Funzionalita'** (piccole, in basso a sinistra)
   - Icona SMS
   - Icona Notifica
   - Icona Scudo/Sicurezza

4. **Illustrazione** (a destra)
   - Casa stilizzata con scudo di protezione
   - Oppure: mockup telefono con schermata app

5. **Sfondo**
   - Gradiente dal colore primario (#006C4C) al secondario (#004D36)
   - Oppure: pattern geometrico sottile

## Checklist Pre-Upload

- [ ] Dimensioni esatte: 1024 x 500 px
- [ ] Formato: PNG 24-bit o JPEG
- [ ] Nessun canale alpha (trasparenza)
- [ ] Dimensione file < 1 MB
- [ ] Testo leggibile su dispositivi piccoli
- [ ] Nessun testo nei bordi (safe area)
- [ ] Colori vivaci e accattivanti
- [ ] Logo/brand visibile
- [ ] Coerenza con icona app

---

## Note Importanti

1. **Safe Area**: Evita testo/elementi importanti nei primi/ultimi 50px laterali

2. **Leggibilita'**: Il testo deve essere leggibile anche in miniatura

3. **Coerenza**: Usa gli stessi colori dell'app (Material Design 3)

4. **Localizzazione**: Se necessario, crea versioni per altre lingue

5. **A/B Testing**: Considera di creare 2-3 varianti per testare quale performa meglio
